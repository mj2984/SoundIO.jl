SoundIO.jl aims to provide a Julia native binding to https://github.com/andrewrk/libsoundio merging the low level access libsoundio provides with the interactivity and speed of development of Julia.

In terms of balance between performance and power, and API convenience, the scale is tipped closer to the former as is the case of the original library but with the features of julia making it more easier to manage while still getting the performance. The Library tries to provide some safety guides but will provide enough low level access for the fine details as required.
